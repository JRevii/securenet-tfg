# External Assets

The original Google Drive folder contains large virtual machine exports:

- `PC1 TFG PIHOLE.ova`
- `PC2 TFG DOCKER .ova`

These files are not stored in GitHub because exported virtual machines are large and may contain system credentials or private configuration.

Recommended storage options:

- Keep them in Google Drive.
- Store checksums in this repository.
- Use GitHub Releases only for sanitised artefacts.
- Use Git LFS if binary tracking is truly required.
