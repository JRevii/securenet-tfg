# Contributing

This is a personal learning and final degree project repository.

Contributions, suggestions and corrections are welcome, especially around:

- Documentation clarity.
- Security best practices.
- Safer configuration examples.
- Monitoring and automation improvements.

## Guidelines

- Do not include real secrets, keys, passwords or tokens.
- Keep examples reproducible and clearly labelled as examples.
- Explain security-sensitive changes.
- Prefer small, focused pull requests.
